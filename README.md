# FamPick - 家庭点单小程序

## 项目介绍

「FamPick」是一款专为家庭成员设计的点单小程序，旨在简化家庭日常餐饮管理，让每个家庭成员都能参与到菜单的制定和选择中。通过直观的界面和丰富的功能，让家庭点餐变得更加便捷和有趣。

## 功能特点

### 1. 多用户登录
- 支持用户名密码登录
- 提供快速登录功能（爸爸、妈妈、爷爷）
- 个性化用户头像

### 2. 今日点单
- 查看当日菜单
- 编辑和管理菜单
- 显示菜品详情、烹饪时间和备注
- 分配菜品主厨

### 3. 菜单浏览
- 分类浏览菜品（家常菜、川菜、粤菜、甜点、汤品）
- 搜索菜品功能
- 查看菜品详情

### 4. 菜品上传
- 上传新菜品信息
- 添加菜品图片
- 填写食材清单和制作步骤
- 分类和难度设置

### 5. 随机点单
- 自定义点单设置（菜品数量、菜系偏好、烹饪时间、难度等级）
- 自动生成随机菜单

### 6. 图像识别
- 拍照或上传图片识别菜品
- 自动生成食材清单和制作步骤
- 保存识别结果到菜单

### 7. 菜品详情
- 查看菜品图片、分类、烹饪时间和难度
- 详细的食材清单
-  step-by-step制作步骤
- 收藏和加入点单功能

## 技术栈

- **前端框架**：纯HTML5 + JavaScript
- **样式框架**：Tailwind CSS v3
- **图标库**：Font Awesome 4.7.0
- **响应式设计**：适配移动端设备

## 如何使用

### 本地运行

1. 克隆或下载项目到本地
2. 直接在浏览器中打开 `index.html` 文件
3. 选择快速登录（爸爸、妈妈、爷爷）或输入用户名密码登录
4. 开始使用各项功能

### 主要操作流程

1. **登录**：选择家庭成员快速登录或输入用户名密码登录
2. **今日点单**：查看当日菜单，点击「+」按钮添加菜品
3. **浏览菜单**：在菜单页面浏览和搜索菜品，点击菜品查看详情
4. **上传菜品**：在上传页面填写菜品信息并保存
5. **随机点单**：在随机点单页面设置偏好并生成随机菜单
6. **图像识别**：在识别页面拍照或上传图片识别菜品

## 设计特点

- **温暖的色彩方案**：使用橙色作为主色调，营造温馨的家庭氛围
- **响应式布局**：适配不同屏幕尺寸的移动设备
- **流畅的动画效果**：页面切换和交互有平滑的过渡效果
- **直观的用户界面**：简洁明了的操作流程，易于上手
- **丰富的视觉反馈**：按钮点击、页面切换等操作有明确的视觉反馈
